## 新地圖 BGM 設計標準格式

新增地圖時，BGM 設計必須輸出以下 5 欄位：

1. **基礎樂理設定** — 調性、BPM、拍號
2. **地貌風格專屬配器** — 流行骨架、主奏管弦、地貌樂器、楓風裝飾
3. **旋律與樂曲結構** — Melody Motif + A/B/C 三段式
4. **音樂情緒與意境** — 具體畫面感描述
5. **AI 生成提示詞** — Style / Instrumentation / Vibe 三欄英文 prompt
